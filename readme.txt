Create and Track custom link forwarders with a minimized URL off your own website.

== Description ==

Create and Track custom link forwarders with a minimized URL off your own website.  Take advantage of short codes, widgets and other features.

== Installation ==

1. Unzip the contents of `wpdevhub-dlm.zip`
2. Upload the full `wpdevhub-dlm` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Access the WPDevHub Link Manager through the links on the left of WordPress.
5. Add, create and interact with links through the provided admin tools.  Widgets and short codes are also available

== Screenshots ==

1. The WPDevHub Link Manager allows you to create an unlimited number of link forwarders that are local to your site.
2. The editor allows you to easily setup and change the directives for your link forwarder.

== Frequently Asked Questions ==

= How can I get help? =

Visit https://www.wpdevhub.com for any and all support inquiries.

= How many links can I create? =

There is no limit to number of links you can create.  Create as many as you are comfortable with.

= How do I change the base /go/ directive? =

Visit the settings page in the navigation bar for the plugin.  There you can specify any forwarding directive that you wish.
